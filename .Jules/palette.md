## 2024-06-07 - Safe button conversions with Tailwind
**Learning:** Tailwind's preflight base CSS resets default `<button>` styles. This allows for safe structural conversions of interactive `<div>` elements (like `.dock-item`) to semantic `<button>` tags without causing visual regressions.
**Action:** Confidently convert interactive `<div>` elements to semantic `<button>` tags in Tailwind projects to improve accessibility without worrying about breaking existing layouts or requiring custom CSS overrides.
